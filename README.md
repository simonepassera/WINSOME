# WINSOME

## Installation

Download : [tar.gz](https://github.com/simonepassera/WINSOME/raw/main/DevWINSOME/560026%20PASSERA%20SIMONE.tar.gz) to exec winsome project.

```bash
tar xzvf 560026\ PASSERA\ SIMONE.tar.gz
```

## Usage

Client :

```bash
cd Winsome
./execClient
```


Server :

```bash
cd Winsome
./execServer
```
